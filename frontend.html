Frontend

Ello es lo primero de tecnología que quise aprender desde el 2020, yo en mis inicios, solamente conocía la etiqueta <p>, no sabía otras etiquetas.
Luego fui investigando más y más, luego me topé con <a>, <img>, <button>, <input>, <head>, <body>, <h1>,<html>, <br>, etc.
Ya posteriormente inicié con CSS, solía ponerlos en el mismo HTML (aún lo hago), inicié con width, height, color, background-color, border, font-family, font-size, border-radius, etc.
Y finalmente con Javascript, recuerdo que la primera vez que vi un código de JS díficil (actualmente ese ejercicio me es recontra fácil), me dije "Ayy, qué complicado, ¿Cómo voy a aprender a escribir esas llaves, paréntesis, function?. Creo que nunca voy a aprender si quiera lo básico de JS, mejor me quedo solo con HTML y CSS",
el código era un ejemplo de como cambiar un texto de un botón usando this.innerHTML y un evento click, luego ya se me fue haciendo más sencillo, continué con los ejemplos de W3Schools de prender y apagar un foquito (cambiar el src de una imagen por medio de 2 botones HTML, DOM), cambiar el color de fondo, el color de texto, el tamaño de texto, o sea, lo básico de manipulación del DOM.

Lenguajes frontend que estuve aprendiendo en esas fechas.

1. HTML

Lenguaje de marcado de hipertexto, el esqueleto de las páginas web.

a) Un button.

2. CSS

Las hojas de estilo de cascada, el diseño de las páginas web.

a) El button ahora tiene otro color.

3. JS

Javascript, lo que hace dinámico una página web.

a) El button que ahora tiene otro color alternará su propio color y su nombre de a azul a amarillo y viceversa.
